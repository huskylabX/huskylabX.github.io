特殊說明:
102.type(0) → 影子類,如人物或怪物的影子,法術等等
102.type(1) → 裝飾品,如聖誕樹,雪人,潘朵拉背後的帳篷,船,蟻洞
102.type(5) → 玩家人物,或不能對話的 NPC,變怪免 CTRL 就是把變怪和玩家的 5 改成 10
102.type(6) → 可以對話 NPC1
102.type(7) → 寶箱,開關
102.tyoe(8) → 可以打開的門
102.type(9) → 可撿取的物品,一般物品,武器等等
102.tyoe(10) → MOB,就是怪物,會出現攻擊符號的?變怪免 CTRL 就是把變怪和玩家的 5 改成 10
102.type(11) → 城牆,城門
102.type(12) → 新的 NPC 可以對話
102.type(14) → 盟屋前的告示牌
102.type(15) → 拍賣告示板

104.attr(1) → 半透明
104.attr(2) → 隱形,可看見名字
104.attr(4) → 正常
104.attr(8) → 發光的
104.attr(9) → 半透明+發光
104.attr(32) → 中毒,綠色
104.attr(128) → 黑邊半透明
104.attr(512) → 石化,灰灰的
104.attr(4096) → 南瓜怪隱形(有影子)
104.attr(4104) → 南瓜怪隱形(沒影子)
104.attr(8192) → 巨人守護者隱形(有影子)
104.attr(8193) → 巨人守護者隱形(沒影子)
